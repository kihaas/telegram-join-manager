# Глоссарий терминов

- **Join‑запрос:** запрос пользователя на вступление в приватный чат или канал, обрабатываемый Telegram API.
- **Капча:** проверка, которую пользователь должен пройти, чтобы бот принял его запрос.
- **FSM (Finite State Machine):** конечный автомат, используемый `aiogram` для хранения состояний пользователей (например, `waiting_captcha`).
- **Redis:** высокопроизводительная in‑memory база данных, используемая как хранилище FSM и, возможно, кэша.
- **SQLAlchemy:** ORM‑фреймворк для работы с реляционной БД (SQLite/PostgreSQL/MySQL).
- **Telegram‑bot:** экземпляр бота, авторизованный через `BOT_TOKEN`.
- **Telegram‑аккаунт:** пользовательский аккаунт Telegram, используемый для join‑запросов (если не только бот).
